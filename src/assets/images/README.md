# Картинки проекта

Кладите сюда все статичные изображения (логотипы, фото, иллюстрации).

## Использование в компонентах

```tsx
import logo from "@/assets/images/logo.png";

<img src={logo} alt="Logo" />
```

Vite сам обработает путь, оптимизирует и добавит хэш.

## Подпапки (по желанию)
- `src/assets/images/cases/` — фото для кейсов
- `src/assets/images/team/` — фото команды
- `src/assets/images/icons/` — svg-иконки

## Альтернатива: папка `public/`
Если нужно, чтобы файл был доступен по фиксированному URL
(например, `/og-image.jpg` для шаринга в соцсетях) — кладите его
в корневую папку `public/` и обращайтесь по абсолютному пути `/file.jpg`.
